What This Measures

This benchmark tracks how well local Ollama models cooperate with Codex for practical development work, not just chat quality.

  • File creation and file editing
  • Terminal command execution and test runs
  • Shell-based HTTP access for web-assisted tasks
  • Dedicated Codex web-search behavior when requested

How To Reproduce

Each run page preserves the raw JSONL traces and summary artifacts so other developers can audit the exact behavior of the model.

  • Run ./codex-local-benchmark.sh --mode native-oss --model <model>
  • Commit the generated docs/ folder to GitHub
  • Enable GitHub Pages for the /docs folder on your default branch

Ownership

Maintained by Pawar Viraj. The site is generated automatically after every benchmark run and is designed to publish cleanly on GitHub Pages.
